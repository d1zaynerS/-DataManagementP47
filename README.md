# 📊 Data Management P47

Система управління даними, написана на **C#**. Проект демонструє роботу з базами даних, використовуючи сучасні підходи та бібліотеки.

### 🛠 Стек технологій
* **Мова:** C# (.NET)
* **ORM:** Entity Framework Core (EF Core)
* **Micro-ORM:** Dapper
* **База даних:** SQL Server / PostgreSQL (обери своє)

### 🚀 Основні можливості
- [x] Інтеграція з базою даних через EF Core для складних зв'язків.
- [x] Швидкі запити за допомогою Dapper.
- [x] Архітектура управління даними та міграції (Migrations).

### ⚙️ Як запустити
1. Склонуйте репозиторій.
2. Налаштуйте рядок підключення (Connection String) у файлі `appsettings.json`.
3. Виконайте команду `Update-Database` у консолі диспетчера пакетів (Package Manager Console).
